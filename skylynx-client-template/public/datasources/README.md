# Datasources

Use this folder for datasource manifests, sample API contracts, and configuration examples that explain where demo data comes from.

This folder is different from `public/mock-data`:

- `mock-data` contains browser-fetchable demo data used at runtime.
- `datasources` contains descriptions, manifests, and sample configs that document how data should be connected or replaced.

Do not commit real credentials or environment-specific connection strings.
