# NorthStar Utilities -- Utility Labor Intelligence Demo

## Vision

Build a portfolio-quality React application that functions as both an
executive presentation and a working AI-enabled utility operations demo.

## Key Decisions

-   Fictional combined electric + gas utility named **NorthStar
    Utilities**.
-   Use user's real electric dataset as the foundation.
-   Generate matching synthetic gas assets, workforce, work orders, and
    AI outputs.
-   Validate all spatial data in QGIS.
-   Present the solution in a React application using the Esri
    JavaScript API.

## Technology

-   React + TypeScript + Material UI
-   Esri JavaScript API
-   QGIS
-   SQL Server or PostgreSQL/PostGIS
-   GeoJSON / File Geodatabase
-   Optional AI assistant and Power BI

## Demo Flow

1.  Welcome
2.  Business Problem
3.  Utility Challenges
4.  NorthStar Overview
5.  Executive Dashboard
6.  Map
7.  Work Orders
8.  AI Crew Recommendation
9.  Explainability
10. Architecture
11. ROI

## Dashboard KPIs

-   Active Work Orders
-   Available Crews
-   Emergency Events
-   Labor Savings
-   Overtime Avoided
-   Travel Reduction
-   Customer Impact

## Workforce

Electric: Trouble, Line, Underground, Transformer, Vegetation. Gas: Leak
Investigation, Emergency Response, Service Installation, Corrosion,
Regulator Maintenance.

## AI Logic

Distance, certifications, travel time, labor cost, overtime, equipment,
priority, business rules.

## Development Plan

Phase 1: Import electric data. Phase 2: Create gas assets. Phase 3:
Build React + Esri JS app. Phase 4: AI recommendations. Phase 5: Polish
presentation.

## Principle

Build a product, not a PowerPoint.
